### Installation üçÄ
Back up your `~/.emacs.d/` first (if you have one):

```
mv ~/.emacs.d ~/.emacs.d.bak
```

Git clone my configuration to your new `~/.emacs.d/` :
```
git clone https://github.com/jarlah/.emacs.d.git ~/.emacs.d
```
